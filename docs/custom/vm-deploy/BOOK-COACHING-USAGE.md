# Book Coaching Guides — How to Use Them

## What You Have

Two coaching frameworks extracted from books, formatted for LLM use:

### 1. Derek Prince — *Blessing or Curse* (283 lines)
**Framework:** Spiritual patterns, generational dysfunction, authority dynamics
**Best for:** Breaking cycles of repeated defeat, family patterns, spiritual blockages
**Key insight:** Some problems aren't solved by trying harder; they need spiritual root work

### 2. Allen Carr — *Easy Way to Quit Emotional Eating* (349 lines)
**Framework:** Addiction deconstruction, fear/belief systems, psychological reframing
**Best for:** Breaking food/comfort cycles, handling emotional triggers without willpower
**Key insight:** Addiction is 99% psychological; remove the illusion and the power disappears

---

## How to Use

### In daily coaching
When the agent detects relevant triggers, it can pull framework-specific guidance:

- Struggling with repeated failure? → Derek Prince framework
- Comfort eating when stressed? → Allen Carr framework

### For specific situations

**Derek Prince applies when:**
- Same family pattern repeating (generational cycles)
- Spiritual blockage feeling (stuck despite effort)
- Authority figures' words affecting you
- Chronic shame despite trying

**Allen Carr applies when:**
- You reach for food when stressed/bored/sad
- You feel trapped by cravings
- Diet willpower isn't working
- You need reframing not restriction
