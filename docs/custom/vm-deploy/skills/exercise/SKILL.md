Prompt: "Say 'done with [workout name]' to complete it in Habitica."

The Habitica daily for the workout matches the day name (e.g., "Monday Workout - Full Body Mobility & Strength Focus").

## RPE Guide (if asked)

| RPE | Meaning |
|-----|---------|
| 3 | Very light, barely any effort |
| 4-5 | Light to moderate, comfortable |
| 6 | Moderate, can talk but focused |
| 7-8 | Hard, last few reps are tough |
| 9-10 | Maximum effort (not used in this plan) |

## Progression

If Henzard consistently completes all sets at the prescribed RPE, suggest:
- Add 1 rep per set, or
- Add a light resistance band, or
- Increase hold time by 5-10 seconds

Never suggest jumping to heavy weights or high-impact exercises without explicit request.
